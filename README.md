Test file for a personal project.

Goal: test several data-driven models within an MPC feedvack loop, largely based on the results in the book "Data Driven Science and Engineering by Brunton and Kutz (https://databookuw.com).

Van der Pol simulations: test uncontrolled methods, such as DMD, eDMD, SINDy and NN-DMD.
Reactor control: similar to the above for a controlled system, based on the reactor model described in Chapter 1 of the book "Model Predictive Control" by Rawlings, Mayne and Diehl.
